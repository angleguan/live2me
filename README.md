# live2me 
> 一个关于个人记录的 APP

> * 也许你是一个喜欢记录自己生活的人，但苦于QQ和微信等社交平台已被五大姑八大姨占领,需要一个记录生活点滴的APP。
> * 也许你是一个想要自律却三分钟热度无法安排好自己生活的人，急需要借助好的一款工具制糊自己。
> * 也许你想有一个完美的记事本。
> * 也许你需要一个优雅的阅读器。
